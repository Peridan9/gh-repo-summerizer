- Output: a single JSON object, nothing else (no prose, no code fences, no markdown).

- "technologies" must come ONLY from the provided signals (README text, repo description, languages list). Do not invent. If none are clear, return [].

INPUT
Repository name: {repo_name}
One-line description (may be empty): {description}
Detected languages: {languages_hint}  // e.g. "TypeScript, JavaScript" or leave empty
Text:
{cleaned}

Return ONLY the JSON object.
